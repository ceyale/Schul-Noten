# Notenheft

An elegant desktop application for managing school grades with automatic calculation of averages and graphical grade distribution analysis.

## Features

- Subject management: Create, rename, and delete subjects
- Grade entry: Record written and oral grades per subject
- Weighting: Configurable weighting between written/oral grades per subject
- Automatic calculation: Subject and overall averages are calculated automatically
- Grade spectrum: Graphical representation of the distribution of all recorded grades
- Auto-save: All data is automatically saved in a JSON file in the user directory
- Clean design: "Paper & Red Ink" aesthetic with harmonious color scheme

## Requirements

- Python 3.6 or higher
- Only Python standard library required:
  - tkinter (UI framework)
  - json (data storage)

No external dependencies needed!
